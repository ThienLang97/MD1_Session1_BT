Đây là nội dung trang web

Đây là 1 ví dụ về cách sử dụng thẻ nav và a để tạo menu trong HTML. Trang web này có menu đơn giản với các liên kết đến trang khác nhau trên trang web. Khi người dùng nhấp vào link này, trang web sẽ chuyển hướng sang trang web đó.
